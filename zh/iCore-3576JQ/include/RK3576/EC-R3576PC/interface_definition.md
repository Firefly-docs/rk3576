# 接口定义

## 整机接口定义

iCore-3576JQ 使用的接口，主要包括：

* 1 x HDMI2.1（最大支持 4K@120Hz 输出）
* 1 x USB3.0 OTG(Type-C)
* 1 x Display Port1.4（最大支持 4K@60Hz 输出）
* 1 x CAN
* 3 x RS232
* 2 x RS485
* 1 x SATA3.0 或 PCIe2.0
* 1 x RELAY（继电器输出）
* 1 x INPUT（光耦隔离输入）
* 3 x LED（一个三色灯，两个单色灯）
* 1 x TF Card
* 1 x USB3.0
* 1 x USB2.0 
* 1 x WIFI
* 1 x Bluetooth
* 2 x RJ45（其中一个支持 1Gbps，一个支持100Mbps）
* 1 × 3.5mm耳机接口
