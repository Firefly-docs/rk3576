# 产品简介

**CQ38W-3576** 配置 Rockchip 八核 64 位 AIOT 处理器 RK3576，采用先进工艺制程，高性能低功耗，内置 ARM Mali G52 MC3 GPU，集成 6 TOPS 算力 NPU，支持 Transformer 架构下大模型的私有化部署；支持 4K@120fps 解码 / 4K@60fps 编码，具备强大的 4K@120fps 高清高帧率显示能力，配置工业级全金属外壳，无风扇被动散热，支持外部看门狗，拥有工业级的稳定性，广泛适用于AI本地部署的应用场景。
