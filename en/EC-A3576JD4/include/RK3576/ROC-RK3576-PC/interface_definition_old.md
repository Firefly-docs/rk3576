# Interface definition

## Interface definition of the whole machine

### EC-A3576JD4 V1.0 interfaces:

* 1 x HDMI2.1 (Max output support 8K@60Hz)
* 1 x Display Port1.4 (Max output support 8K@30Hz)
* 1 x USB3.0 OTG(Type-C)
* 2 x MIPI-DSI
* 1 x MIPI-CSI
* 1 x SATA3.0 or PCIe2.0
* 1 x TF Card
* 1 x USB3.0
* 3 x USB2.0 (Two USB2.0 are led by a 20-pin pin row)
* 1 x WIFI
* 1 x Bluetooth
* 2 x RJ45 (Suport 1Gbps)
* 1 × 3.5mm Headphone (<font color=#FF0000>V1.0 version does not support recording</font>)
* 1 × Mic（2P-1.25mm）
